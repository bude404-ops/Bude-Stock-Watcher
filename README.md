# 🔥 BudE Leverage Radar

Mobile-first leverage trading assistant — 100% free, no API keys.

## Features
- Real-time 3x ETF tracking (TQQQ, SQQQ, UPRO, SPXU, SOXL, etc.)
- Leverage risk score + position sizer
- Decay warnings & cascade detection
- Sector heatmap with leverage tips

## 100% Free
- Uses Yahoo Finance (no API key)
- GitHub Pages hosting
- LocalStorage watchlist

## Setup
1. Fork/clone repo
2. Enable GitHub Pages (Settings → Pages → main branch → /root)
3. Visit your URL
4. Done
