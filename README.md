# 🔥 BudE Leverage Radar

Mobile-first leverage trading assistant — heatmap, risk scanner, position sizer, and decay warnings.

## Features
- **Real-time 3x ETF tracker** (TQQQ, SQQQ, UPRO, SPXU, SOXL, etc.)
- **Leverage risk score** — VIX + market internals = safe or danger
- **Position sizer** — calculates max position based on account size, risk %, stop loss, and leverage multiplier
- **Decay warnings** — alerts when choppy markets erode leveraged ETF value
- **Liquidity cascade detector** — early warning system for market stress
- **Sector heatmap** — find trending sectors for directional leverage plays
- **Action plans** — tells you exactly what to do based on risk level

## 100% Free
- No API keys needed
- Uses Yahoo Finance (free)
- GitHub Pages hosting (free)
- LocalStorage for watchlist

## Setup
1. Create repo on GitHub
2. Add all files above
3. Enable GitHub Pages (Settings → Pages → main branch)
4. Visit `https://yourusername.github.io/Bude-Stock-Watcher`
